## 二、命令片段管理

### 2.1 核心功能

- 创建/编辑/删除命令片段
- 支持变量替换（`<service>`、`<port>` 等）
- 分类和标签管理
- 快捷键快速调用（Cmd/Ctrl + J）
- 共享到团队，支持评论和版本历史

## 四、凭证管理

### 4.1 核心功能

- 服务器账号/数据库账号/SSH 密钥/API 密钥存储
- 加密存储（端到端加密，AES-256）
- 共享到团队，权限分级
- 审计日志（谁访问/复制/修改）
- 密钥轮换机制
- 双因素认证支持

### 4.2 凭证类型

| 类型 | 数量 | 说明 |
|-----|------|------|
| 服务器账号 | 23 | SSH 登录凭证 |
| 数据库账号 | 12 | MySQL/Redis 等 |
| SSH 密钥 | 8 | 私钥/公钥对 |
| API 密钥 | 5 | 第三方服务 |
| 第三方服务 | 3 | AWS/GCP 等 |

### 4.3 安全机制

```
加密存储：端到端加密 (E2EE) + AES-256 + 密钥分离
访问控制：角色权限分级 + 细粒度权限 + IP 白名单
审计日志：完整记录所有访问/复制/修改操作
密钥轮换：定期提醒 + 离职触发 + 安全事件触发
双因素认证：支持 TOTP/短信/硬件密钥
```

## 五、实时监控

### 5.1 核心功能

- CPU/内存/磁盘/网络实时监控
- 历史趋势图表（24 小时）
- 告警设置和通知
- 团队监控共享
- 告警升级机制

### 5.2 监控指标

| 指标 | 刷新频率 | 告警阈值 |
|-----|---------|---------|
| CPU | 5s | > 80% 持续 5 分钟 |
| 内存 | 5s | > 90% |
| 磁盘 | 60s | > 85% |
| 网络 | 5s | > 10 MB/s |

## 六、团队协作

### 6.1 团队管理

- 创建/管理团队
- 成员邀请/移除
- 角色权限（管理员/高级成员/普通成员/访客）
- 子团队支持

### 6.2 共享内容

| 内容类型 | 说明 |
|---------|------|
| 会话配置 | 服务器连接配置共享 |
| 命令片段 | 团队命令模板库 |
| 文件 | 团队文件共享目录 |
| 凭证 | 团队账号密码共享 |
| 监控视图 | 团队监控面板 |
| 主题 | 团队统一主题 |

### 6.3 权限模型

```
管理员：全部权限（查看/编辑/删除/分享/执行）
高级成员：查看/编辑/执行
普通成员：查看/执行
访客：仅查看
```
