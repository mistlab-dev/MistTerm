## 一、核心功能清单

| 功能模块 | 个人功能 | 团队协作 |
|---------|---------|---------|
| 命令片段 | 创建/管理/调用 | 共享/评论/版本 |
| SFTP 传输 | 上传下载/断点续传 | 团队文件/权限 |
| 实时监控 | CPU/内存/磁盘/网络 | 团队监控/告警 |
| 云端同步 | 个人配置同步 | 团队配置同步 |
| 凭证管理 | 个人凭证存储 | 团队凭证共享 |
| 主题定制 | 个人主题 | 团队主题共享 |

## 二、命令片段管理

### 2.1 核心功能

- 创建/编辑/删除命令片段
- 支持变量替换（<service>、<port> 等）
- 分类和标签管理
- 快捷键快速调用（Cmd/Ctrl + J）
- 共享到团队，支持评论和版本历史

## 三、SFTP 文件传输（含团队协作）

### 3.1 功能概述

**SFTP** 提供本地与远程服务器之间的文件传输，支持拖拽、批量操作、断点续传。支持**团队文件共享**和**协作传输**，实现团队文件统一管理。

### 3.2 核心功能

#### 3.2.1 文件传输界面（含团队文件）

```
┌───────────────────────────┬─────────────────────────────────────────┐
│  📁 本地 (Mac)             │  📁 远程 (ubuntu@124.220.224.223:~)     │
│  ──────────────────────── │  ───────────────────────────────────── │
│  🏠 桌面                   │  ~                                      │
│  📁 Documents              │  ├─ 📁 nginx                            │
│  📁 Projects               │  │  └─ 📄 nginx.conf (团队共享)        │
│  📁 团队共享               │  ├─ 📁 app                             │
│  │  ├─ 📁 研发一组         │  │  └─ 📄 package.json (团队共享)      │
│  │  └─ 📁 公共资源         │  └─ 📄 config.yaml (🔐 需权限)         │
│                           │                                         │
│  [📤 上传] [📥 下载] [📤 上传到团队] [💬 评论]                      │
└───────────────────────────┴─────────────────────────────────────────┘
```

#### 3.2.2 团队文件共享

```
┌─────────────────┬───────────────────────────────────────────┐
│  🔍 搜索文件...   │  📁 研发一组 (23 个文件，156 MB)           │
│                 │  ├─ 📁 部署脚本                            │
│  📁 研发一组     │  │  ├─ 📄 deploy.sh (李明 10 分钟前)       │
│  📁 运维组       │  │  └─ 📄 rollback.sh (张光宇 2 天前)      │
│  📁 公共库       │  ├─ 📁 配置文件                           │
│                 │  │  └─ 📄 nginx.conf (王芳 1 小时前)       │
```

#### 3.2.3 文件协作功能

```
文件协作场景:
├─ 配置文件协作：上传/修改 → 通知成员 → 在线预览评论 → 版本回滚
├─ 部署脚本共享：上传到团队 → 直接使用 → 执行记录可查 → 版本审批
├─ 日志归档共享：自动归档 → 搜索下载 → 权限控制 → 保留策略
└─ 批量文件传输：多服务器 → 共享凭证 → 统一管理 → 失败重试
```


### 3.4 验收标准

```
✅ 基础功能: 文件浏览、上传下载、批量传输、拖拽、递归传输
✅ 团队协作: 团队文件共享、权限管理、评论反馈、版本历史、更新通知、批量传输
✅ 高级功能: 断点续传、队列管理、文件操作、搜索、预览
✅ 性能要求: GB 级传输、速度准确、队列流畅、恢复<5s、同步<3s
```

```
├─ 快速连接服务器：选择会话 → 自动填充账号凭证 → 无需手动输入
├─ 批量操作：选择多个服务器 → 使用共享凭证批量执行 → 统一认证
├─ 新人入职：加入团队 → 自动获得凭证访问权限 → 无需逐个申请
└─ 离职交接：离职人员权限自动回收 → 凭证自动轮换 → 审计日志完整
```

#### 6.2.5 凭证安全机制

```
安全机制:
├─ 加密存储：端到端加密 (E2EE)、AES-256、密钥分离、云端仅存密文
├─ 访问控制：角色权限分级、细粒度权限、时间窗口、IP 白名单
├─ 审计日志：谁在什么时候访问/复制/修改了凭证，不可篡改
├─ 密钥轮换：定期提醒、离职触发、安全事件触发、历史可追溯
└─ 双因素认证：敏感操作需 2FA、支持 TOTP/短信/硬件密钥
```
