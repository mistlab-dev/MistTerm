## 🏗️ 系统架构（待完善）

### 技术栈

| 层级 | 技术选型 | 说明 |
|-----|---------|------|
| 前端 | Electron + React | 跨平台桌面应用 |
| 终端 | xterm.js | 终端渲染 |
| SSH | node-ssh / libssh | SSH 连接 |
| SFTP | ssh2-sftp-client | SFTP 传输 |
| 数据存储 | SQLite + 加密 | 本地加密存储 |
| 云端同步 | 自研 API | 配置同步 |
