> 版本：v1.0  
> 原型地址：`docs/product/proto-terminal-embedded.html`  
> 技术栈建议：Electron / Tauri + React + xterm.js  
> 更新日期：2026-04-30

## 一、整体结构

窗口采用**三层布局**，从上到下依次为：

```
┌────────────────────────────────────────────────────────┐
│  标题栏 (Title Bar)                                    │
├────────────────────────────────────────────────────────┤
│  ┌──────────┐  ┌──────────────────┐  ┌──────────────┐ │
│  │ 左侧面板  │  │    终端区域      │  │  右侧面板    │ │
│  │ 连接管理  │  │   (Terminal)    │  │  命令片段    │ │
│  │ (可折叠)  │  │  Tabs + 输出+   │  │  (可折叠)    │ │
│  │           │  │  输入行         │  │              │ │
│  └──────────┘  └──────────────────┘  └──────────────┘ │
├────────────────────────────────────────────────────────┤
│  状态栏 (Status Bar)                                    │
└────────────────────────────────────────────────────────┘
```

### 层级说明

| 层 | 高度 | 内容 |
|---|---|---|
| 标题栏 | 10px padding | 红绿灯控制点 · 应用名称 · SSH连接时长 |
| 主体 | flex:1 (自适应) | 三栏等分布局：左侧面板 / 终端 / 右侧面板 |
| 状态栏 | 4px padding | 复原按钮 · 连接信息 \| 快捷工具 · 统计 |

---

## 二、标题栏 (Title Bar)

### 视觉
- 背景：`#13131c`
- 底部边框：`1px solid rgba(255,255,255,0.04)`
- 居中显示应用名称 `MistTerm`，字号 13px，透明度 30%
- 右侧显示连接信息，如 `SSH · 2h 34m`，字号 11px，透明度 20%

### 交互
- 仅作为窗口标识，无操作功能（窗口控制由操作系统提供）

---

## 三、左侧面板 — 连接管理 (Sessions)

### 默认状态
- 宽度：200px，圆角 6px，半透明暗色背景
- **标题行**：`连接`（大写英文，字号 10px）+ `−` 收起按钮
- **搜索输入框**：`🔍 搜索连接…`，字号 11px，半透明
- **分类标签**：全部 / 在线 / 离线（等分，字号 10px，hover 高亮）
- **连接列表**：每个条目包含图标 + 名称 + 连接时长

### 连接条目
| 示例 | 图标 | 名称 | 副信息 |
|---|---|---|---|
| 当前激活 | 🖥 | 生产服务器 | `2h 34m`（绿色） |
| 在线 | 🖥 | 测试服务器 | `45m`（绿色） |
| 离线 | 🖥 | 数据库 | `离线`（灰色） |

### 折叠/展开
- 触发：点击标题栏 `−` 按钮
- **折叠动画**：宽度渐变为 0，完全隐藏，`transition: all 0.2s ease`
- 折叠后 → 状态栏左侧显示 `▸ 连接 · 3`（淡紫色可点击按钮）
- 点击 `▸ 连接 · 3` → 面板展开，按钮消失

---

## 四、终端区域 (Terminal)

### Tab 栏
- 背景：`rgba(255,255,255,0.02)`，底部边框分割
- 每个 tab：名称 + 绿色圆点（连接状态） + 关闭按钮（hover 出现）
- 激活 tab：文字 80% 亮度，背景 `#0a0a12`
- 末尾 `+` 按钮新建 tab

### 终端输出区
- 字体：JetBrains Mono，字号 13px，行高 1.7
- 深色背景 `#0a0a12`
- **语法高亮**：
  - `➜` 提示符：绿色 `#4CAF50`
  - 路径：紫色 `#667eea`
  - 命令：白色 90% 亮度
  - 输出：白色 40% 亮度
- **输入方式**：输出区最后一行为当前输入行（`contenteditable`），无独立输入框
- 滚动条：4px 宽，半透明

### 设计要点
- 终端是**统一的可滚动区域**，输出和输入在同一视图中
- 没有独立的输入行/建议条/工具栏——它们被剥离到右侧面板和状态栏
- 命令自动建议可通过右侧片段面板搜索，或由 Tab 激活

---

## 五、右侧面板 — 命令片段 (Fragments)

### 默认状态
- 宽度：260px
- **标题行**：`命令片段` + `−` 收起按钮
- **搜索输入框**：`🔍 搜索片段…`
- **分类标签**：常用 / Docker / K8s / 全部

### 片段卡片
每个卡片包含：

```
┌─────────────────────────┐
│ 查看 Pod 日志     [团队] │ ← 标题 + 标签
│ kubectl logs -f ...     │ ← 命令原文 (单行截断)
│ 320次 · 95%成功 · 1.2s  │ ← 统计 (使用/成功率/耗时)
└─────────────────────────┘
```

#### 标签系统
| 标签 | 颜色 | 场景 |
|---|---|---|
| 团队 | 绿色 `#4CAF50` | 团队共享命令 |
| 个人 | 紫色 `#667eea` | 个人收藏 |
| 模板 | 橙色 `#FF9800` | 预制模板 |

### 折叠/展开
- 同左侧面板，折叠后状态栏右侧显示 `▸ 命令片段 · 5`
- 点击恢复展开

---

## 六、状态栏 (Status Bar)

### 布局
```
[▸ 连接 · 3] │ ⚡ ubuntu@prod-server-01        │ [▸ 命令片段 · 5] 📋📤🔍📊 · 1,234次 ↑8%
```

### 左侧
| 元素 | 说明 |
|---|---|
| `▸ 连接 · N` | 左面板收起时显示，淡紫色可点击 |
| `⚡ ubuntu@prod-server-01` | 当前活跃连接信息 |

### 右侧
| 元素 | 说明 |
|---|---|
| `▸ 命令片段 · N` | 右面板收起时显示，淡紫色可点击 |
| 📋 | 复制当前命令 (hover 提示) |
| 📤 | 上传文件 |
| 🔍 | 搜索终端输出 |
| 📊 | 统计面板 |
| `·` | 分隔符 |
| `1,234次 ↑8%` | 本周命令使用统计 |

### 交互
- **`▸` 按钮**：仅在对应面板折叠时可见，点击展开面板并自动隐藏
- **工具按钮**：hover 到 25% 透明度，点击触发对应操作
- **分隔符**：`·` 和 `｜` 极低透明度，不干扰阅读

## 七、交互逻辑汇总

| 操作 | 效果 |
|---|---|
| 点击标题 `−` | 对应面板折叠消失 |
| 点击状态栏 `▸ 名称 · N` | 对应面板展开还原 |
| 点击片段卡片 | 将命令填入终端输入区 |
| 点击片段分类标签 | 按分类过滤片段列表 |
| 搜索输入框 | 实时过滤连接 / 片段 |
| hover Tab 关闭按钮 | 关闭按钮显现 |
| 点击 `+` Tab | 新建终端会话 |

## 九、实现建议

### 技术选型
- **桌面壳**：Electron (成熟) 或 Tauri (更轻量)
- **终端引擎**：xterm.js + node-pty
- **UI 框架**：React 或 Vue，无需重型组件库
- **面板动画**：CSS transition + max-width 控制折叠

### 数据结构 (前端)
```typescript
interface Connection {
  id: string;
  name: string;
  host: string;
  status: 'online' | 'offline';
  connectedAt?: number;
  sessionId?: string;
}

interface Fragment {
  id: string;
  title: string;
  command: string;
  tag: 'personal' | 'team' | 'market';
  category: string; // 'common' | 'docker' | 'k8s' | 'all'
  usageCount: number;
  successRate: number;
  avgTime: number;
}

interface Tab {
  id: string;
  title: string;
  sessionId: string;
  active: boolean;
}
```

### 状态管理
- 面板折叠状态：`collapsedPanels: { left: boolean, right: boolean }`
- 连接列表：`Connection[]`
- 片段列表：`Fragment[]`
- 当前分类筛选：`activeCategory: string`
- 终端 tabs：`Tab[]` + `activeTabId`
