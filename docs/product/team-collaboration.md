## 三、团队协同（核心亮点）

### 3.1 功能说明

团队协同是 MistTerm 的**核心亮点**，将**命令片段管理**与**团队协作**深度融合，打造团队知识沉淀和共享的平台。不仅支持个人命令片段，更支持团队共享、协作编辑、版本控制、权限管理和审计追踪。

### 3.2 核心价值

```
🚀 效率提升：团队常用命令一次共享，全员复用
📚 知识沉淀：新人快速上手，避免重复造轮子
🔒 安全可控：权限分级，操作可追溯
🤝 协作无间：评论讨论，持续优化
```

### 3.4 命令片段协作

#### 片段类型

| 类型 | 说明 | 可见范围 | 编辑权限 |
|-----|------|---------|---------|
| 个人片段 | 仅自己可见和使用 | 仅自己 | 仅自己 |
| 团队片段 | 团队成员可见 | 指定团队 | 创建者 + 高级成员 |
| 团队模板 | 团队官方模板 | 指定团队 | 管理员 |
| 公开模板 | 全员可见（可选） | 所有团队 | 仅管理员 |

#### 协作流程

```
创建片段 → 选择共享范围 → 团队成员可见 → 评论/建议 → 协作编辑 → 版本历史
    ↓
   个人片段（默认）
    ↓
   共享到团队 → 选择团队 → 设置权限
    ↓
   团队成员收到通知 → 查看/评论/建议修改
    ↓
   创建者审核建议 → 接受/拒绝 → 生成新版本
    ↓
   版本历史记录 → 可回滚到任意版本
```

#### 版本控制

| 版本类型 | 说明 | 自动触发 |
|---------|------|---------|
| 主版本 (v1.0) | 重大变更 | 手动发布 |
| 次版本 (v1.1) | 功能更新 | 手动发布 |
| 修订版本 (v1.1.1) | 小修小改 | 自动保存 |

**版本对比：**
```
v1.0 → v1.1
────────────────────────────────
- 命令：docker restart <container>
+ 命令：docker restart -t <timeout> <container>

变更说明：增加超时参数支持
变更人：张三
变更时间：2026-04-24 08:30
```

### 3.5 权限模型

#### 角色定义

| 角色 | 权限说明 | 适用人群 |
|-----|---------|---------|
| 管理员 | 全部权限（创建/编辑/删除/分享/审核/删除评论） | 团队负责人 |
| 高级成员 | 创建/编辑/删除自己的片段，评论，执行 | 资深成员 |
| 普通成员 | 查看/执行/评论，不能编辑他人片段 | 普通成员 |
| 访客 | 仅查看，不能执行和评论 | 外部人员 |

#### 权限矩阵

| 操作 | 管理员 | 高级成员 | 普通成员 | 访客 |
|-----|-------|---------|---------|------|
| 查看片段 | ✅ | ✅ | ✅ | ✅ |
| 执行片段 | ✅ | ✅ | ✅ | ❌ |
| 创建片段 | ✅ | ✅ | ❌ | ❌ |
| 编辑自己的片段 | ✅ | ✅ | ✅ | ❌ |
| 编辑他人的片段 | ✅ | ❌ | ❌ | ❌ |
| 删除自己的片段 | ✅ | ✅ | ✅ | ❌ |
| 删除他人的片段 | ✅ | ❌ | ❌ | ❌ |
| 发表评论 | ✅ | ✅ | ✅ | ❌ |
| 删除评论 | ✅ | 自己的 | ❌ | ❌ |
| 邀请成员 | ✅ | ❌ | ❌ | ❌ |
| 管理权限 | ✅ | ❌ | ❌ | ❌ |
| 查看审计日志 | ✅ | ❌ | ❌ | ❌ |

### 3.7 模板市场（亮点功能）

#### 概念

模板市场是团队协同的延伸，允许团队将优质命令片段发布到公共市场，供其他团队发现和采用。

#### 分类

```
📦 运维类
  ├─ 服务器管理
  ├─ 网络配置
  ├─ 监控告警
  └─ 备份恢复

📦 开发类
  ├─ 环境搭建
  ├─ 构建部署
  ├─ 数据库操作
  └─ 日志分析

📦 安全类
  ├─ 漏洞扫描
  ├─ 安全加固
  └─ 审计日志

📦 工具类
  ├─ 文件操作
  ├─ 文本处理
  └─ 系统工具
```

#### 发布流程

```
1. 团队选择优质片段
2. 提交审核（格式规范、安全性检查）
3. 审核通过 → 发布到市场
4. 其他团队浏览/搜索/采用
5. 使用统计和反馈
```

#### 采用流程

```
浏览市场 → 搜索/分类 → 查看详情 → 预览命令 → 一键采用
    ↓
  采用到个人/团队片段库
    ↓
  可选：同步更新（原作者更新时提示）
```

### 3.8 审计和日志

#### 审计内容

| 事件类型 | 记录内容 | 保留期限 |
|---------|---------|---------|
| 片段创建 | 创建人、时间、内容、团队 | 永久 |
| 片段编辑 | 编辑人、时间、变更内容、版本 | 永久 |
| 片段删除 | 删除人、时间、片段信息 | 永久 |
| 片段执行 | 执行人、时间、服务器、结果 | 1 年 |
| 评论发布 | 评论人、时间、内容 | 永久 |
| 权限变更 | 操作人、时间、变更详情 | 永久 |
| 成员加入/退出 | 操作人、时间、成员信息 | 永久 |

### 3.10 交互说明

| 交互 | 说明 |
|-----|------|
| 创建团队片段 | 创建时选择「共享到团队」，选择目标团队 |
| 协作编辑 | 高级成员可编辑团队片段，生成新版本 |
| 评论讨论 | 任何有权限的成员可评论，作者可回复 |
| 版本回滚 | 管理员可回滚到任意历史版本 |
| 采用模板 | 从模板市场一键采用到个人/团队库 |
